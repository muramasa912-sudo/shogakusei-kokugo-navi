# iPad/WEB版 使い方

## はじめに

iPad/WEB版は、同梱の `web-ipad.zip` をWebサーバーへ配置して利用します。iPadではSafariで開き、ホーム画面へ追加するとアプリに近い形で起動できます。

> iPad/WEB版は、最初の1回は配布元ページまたはローカルサーバーへのアクセスが必要です。一度必要なファイルを読み込んだ後、Service Workerのキャッシュにより、機内モードでも利用できることを目指しています。

## Safariで開く

1. `web-ipad.zip` を展開し、Webサーバーで公開します。
2. iPadのSafariで配布先のURLを開きます。
3. ホーム画面、学年選択、本文、問題、解説が表示されることを確認します。

パソコンでのローカル確認例:

```powershell
cd web-ipad
python -m http.server 8080
```

同じネットワーク内のiPadから、パソコンのIPアドレスとポートを指定して開きます。`localhost` はiPad自身を指すため、別の端末から使う場合はパソコンのIPアドレスを使います。

## ホーム画面へ追加

1. Safariの共有ボタンを開きます。
2. 「ホーム画面に追加」を選びます。
3. 名前を確認して「追加」を選びます。
4. 作成された「国語ナビ」アイコンから起動します。

表示名はiPadOSのバージョンにより異なる場合があります。

## 機内モードの確認

1. オンラインの状態でアプリを一度開き、教材画面まで進みます。
2. Safariを閉じます。
3. iPadを機内モードにします。
4. ホーム画面のアイコンから開き、教材一覧、本文、問題、解説、先生画像を確認します。

初回のキャッシュが完了していない場合は利用できません。その時は一度インターネットへ接続し、必要な画面を開いてください。

## 読み上げ機能

ブラウザ標準の読み上げ機能を使います。本文、問題、選択肢、解説、先生コメント、学習通知表を読み上げられます。端末の音声機能が利用できない時は、画面の案内に従ってください。外部の音声サービスは使いません。

## データ保存

よび名、学習履歴、定期テスト結果、設定、学びの木はブラウザ内の保存領域に保存されます。別の端末や別のブラウザへ自動で同期されません。SafariのWebサイトデータを消去すると、保存内容も消える場合があります。

## 更新時の注意

Web版を更新した後に古い画面が表示される場合は、Safariを閉じてから再度開いてください。改善しない場合は、Wi-Fi接続中に配布先を開き直し、必要に応じてSafariのWebサイトデータを確認してください。データ消去前には、必要な学習記録が端末内保存であることを理解してください。

## 動かない時の対処

- 配布先のURLをSafariで開けるか確認する。
- 初回は通信可能な状態で開く。
- Webサーバーが `index.html` を配信しているか確認する。
- ブラウザの保存領域が不足していないか確認する。
- 読み上げだけが使えない場合は、iPadの音声設定を確認する。
