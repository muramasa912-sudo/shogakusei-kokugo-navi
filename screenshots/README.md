## 撮影時の確認

- Android版は縦画面で本文とボタンが読めることを確認します。
- iPad版はSafariで開いた画面と、ホーム画面に追加した後のアイコンを確認します。
- 先生画像、読み上げボタン、学びの木、スキル別グラフが見切れていないことを確認します。
- 学習者のよび名を写す場合は、ニックネームまたは撮影用の表示にしてください。
