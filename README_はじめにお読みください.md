# 小学生国語ナビ v1.0

小学生国語ナビは、小学生が「本文に戻る」「根拠を探す」「もう一度考える」力を育てるための国語読解アプリです。
正解数だけでなく、本文を読み直したこと、最後まで取り組んだこと、考え直したことも大切にします。

## 対象学年

小学校1年生から6年生までを対象にしています。低学年は短い本文とやさしい言葉を中心にし、高学年は理由・根拠・要約・主題・選択肢の違いを考える学習へ段階的に進めます。

## 主な機能

- やさしい読解、基礎復習、論理読解、名作文庫、定期テスト対策、思考力強化
- 本文・問題・選択肢・解説の読み上げ
- 先生キャラクターによる、責めないコメント
- 「本文をもう一度読む」「もう一度考える」導線
- 正解だけでなく学び方も記録する学びの木、バッジ、ことばの花
- 学習通知表、保護者レポート、スキル別のようす
- 小1〜小4向け本文の読みやすいルビ表示

## Android版とiPad/WEB版

- **Android版**: `Android版` フォルダのAPKを端末へインストールして利用します。教材と先生画像はAPKに同梱されています。
- **iPad/WEB版**: `iPad_WEB版` のWeb版をSafariなどで配信して開きます。ホーム画面に追加すると、アプリに近い形で使えます。

詳しい手順は、それぞれのフォルダ内の説明書をご覧ください。

## オフライン仕様

Android版は、教材・画像・設定保存を端末内で扱う設計です。アプリ本体として外部通信を必要とせず、AndroidManifestにもINTERNET権限を含みません。

iPad/WEB版は、最初の1回は配布元ページまたはローカルサーバーへのアクセスが必要です。一度必要なファイルを読み込んだ後は、Service Workerのキャッシュにより、機内モードでも利用できることを目指しています。端末やSafariの保存状況によっては、初回の読み込みが完了していることを確認してください。

## よび名とプライバシー

初回に入力するよび名は、本名でなくても構いません。ニックネームでも使えます。よび名、学習履歴、設定は端末内またはブラウザ内の保存領域に保存され、アプリから外部送信しません。設定画面から変更・削除できます。

## 青空文庫教材について

一部の中学年・高学年教材は青空文庫を題材にしています。必要な出典情報は教材JSONに保持しています。小1〜小2には難しい青空文庫教材を表示しない設計です。アプリの起動時に青空文庫へアクセスすることはありません。

## 思考力強化モードについて

思考力強化モードは、根拠・要約・主題・選択肢比較・記述の骨組みを考える高学年向けのモードです。最難関中学レベルを意識した完全オリジナル問題を使っており、実在の入試問題や市販教材は使用していません。

## おすすめの始め方

1. 初めて使う子は「やさしい読解」から始めます。
2. 読むことに苦手意識がある子は「基礎復習コース」で短い本文をゆっくり確認します。
3. 学校のテスト前は「定期テスト対策」を使います。
4. 高学年で深く考える練習をしたい時は「論理読解」や「思考力強化モード」に進みます。
