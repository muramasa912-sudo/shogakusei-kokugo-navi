# オフライン仕様について

## Android版

Android版は、教材JSON、先生画像、アプリアイコンをAPKに同梱しています。AndroidManifestにはINTERNET権限を含めていません。

Android版は、アプリ本体として外部通信を必要としない設計です。教材の閲覧、問題への回答、解説、先生コメント、学びの木、学習履歴、保護者レポートは端末内で扱います。

読み上げは端末標準のTextToSpeechを使用します。端末に日本語音声が用意されていない場合は読み上げを利用できないことがありますが、教材の学習自体は続けられます。

## iPad/WEB版

iPad/WEB版は、教材JSON、先生画像、JavaScript、CSS、アイコンをWeb版の配布物に同梱しています。外部CDN、外部API、外部フォントは使いません。

Service Workerが必要なファイルをキャッシュします。最初に開く時は、配布元ページまたはローカルサーバーからファイルを取得する必要があります。初回読み込み後は、端末・ブラウザのキャッシュ状況が保たれていれば、機内モードでも利用できることを目指しています。

## 外部通信について

Android版は、アプリ本体として外部通信を必要としない設計です。

iPad/WEB版は、初回読み込み時にファイルを取得する必要がありますが、外部APIや外部CDNに依存しない構成です。

青空文庫に関するURLは出典情報として教材JSONに保持しています。アプリ実行時に青空文庫から本文や問題を取得することはありません。

## 保存される情報

### Android版

よび名、学習履歴、定期テスト結果、設定、学びの木の進捗は端末内のアプリ保存領域に保存されます。

### iPad/WEB版

よび名、学習履歴、定期テスト結果、設定、学びの木の進捗はブラウザ内の保存領域に保存されます。SafariのWebサイトデータを削除すると、これらの記録も消える場合があります。

どちらの版も、入力したよび名や学習記録をアプリから外部送信する設計ではありません。
