# 思考力強化モードについて

## 目的

思考力強化モードは、高学年向けに、本文の根拠・要約・主題・選択肢比較・記述の骨組みを考える練習をするモードです。難しい問いに向き合う時も、正解だけでなく、どこで迷ったかを解説で確かめることを大切にします。

## 取り組む力

- 本文の根拠を示す力
- 理由と結果をつなぐ力
- 文章全体を短くまとめる力
- 主題を考える力
- 選択肢の違いを比べる力
- 記述で答える時の骨組みを作る力

## 教材について

最難関中学レベルを意識した完全オリジナル問題です。実在の入試問題・市販教材は使用していません。

このモードは、入試の結果を約束するものではありません。難しい問題に取り組み、解説から考え方を学ぶための学習補助です。

## 使い方

1. 時間を意識しながら本文と選択肢を読みます。
2. 迷った時は、本文のどこが根拠になるかを探します。
3. 解説で、考え方の手順と選択肢の見分け方を確認します。
4. 記述の骨組みがある時は、根拠、考え方、まとめの順で考えます。

高得点を急ぐより、解説を読んで次に見るべきポイントを一つ持ち帰ることを大切にしてください。
