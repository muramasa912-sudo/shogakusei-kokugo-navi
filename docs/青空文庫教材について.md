# 青空文庫教材について

小学生国語ナビでは、一部の中学年・高学年教材に青空文庫を題材とした教材を使用しています。

## 出典情報

教材JSONには、作品の出典に関する情報を保持しています。出典表示が必要な画面では、青空文庫由来であることを確認できるようにしています。

## 学年に合わせた扱い

- 小1〜小2には、難しい青空文庫教材を表示しません。
- 小3〜小4では、読みやすい場面を短く区切った中学年向け教材を使います。
- 高学年では、古い言い回しや深い心情を、文脈から考える読解素材として扱います。

## 旧語・古い言い回し

古い言い回しが含まれる教材では、先生コメントや解説で必要に応じてやさしい言い換えを示します。すべてを暗記するのではなく、前後の文や場面から意味を考えることを大切にします。

## オフライン利用

青空文庫由来の教材も、アプリに同梱した教材JSONから表示します。アプリの起動時や学習中に青空文庫へアクセスして本文や問題を取得することはありません。
