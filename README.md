# 小学生国語ナビ Web版（iPad Safari向け）

このフォルダは、Android版の教材JSONを流用したiPad Safari向けWebアプリ版です。iPadのSafariで開き、ホーム画面に追加すると、アプリのように起動できます。

## ローカル確認

プロジェクト直下から次を実行します。

```powershell
cd web-ipad
python -m http.server 8080
```

ブラウザで次を開きます。

```text
http://localhost:8080
```

## iPad Safariで開く方法

1. Web版を配信しているURLをiPad Safariで開く
2. 画面が表示されたら、一度教材一覧や本文を開く
3. 共有ボタンから「ホーム画面に追加」を選ぶ
4. ホーム画面の「国語ナビ」アイコンから起動する

ホーム画面追加時には「国語ナビ」アイコンが表示されます。アイコン画像は `web-ipad/assets/icons/` に同梱されています。

ホーム画面追加やService Workerによるキャッシュを確実に使う配信では、HTTPSのURLを使ってください。`localhost` は開発確認用です。

## オフライン利用

初回アクセス時は、教材JSON、先生画像、画面ファイルを読み込むため通信が必要です。初回読み込み後はService Workerのキャッシュにより、機内モードでも主要機能を使えることを目指しています。

Web版アイコンもService Workerでキャッシュされるため、オフライン時もホーム画面やブラウザ上で表示されます。

キャッシュ状況は端末やSafariの状態に左右されます。教材が表示できない場合は、もう一度インターネットに接続してWeb版を開いてください。

## 外部通信について

Web版の画面、教材、画像、JS、CSSは `web-ipad/` 内のローカルファイルを使います。外部CDN、外部フォント、外部APIは使いません。

青空文庫のURLは出典情報として教材JSON内に保持していますが、Web版の実行時に青空文庫へ本文を取りに行く処理はありません。

## 教材

Web版は次の教材JSONを読み込みます。

- `assets/lessons/kokugo_sample_lessons.json`
- `assets/lessons/kokugo_lower_grade_expansion_lessons.json`
- `assets/lessons/kokugo_middle_grade_expansion_lessons.json`
- `assets/lessons/aozora_middle_grade_lessons.json`
- `assets/lessons/aozora_upper_grade_lessons.json`
- `assets/lessons/thinking_challenge_lessons.json`

教材ID、問題ID、正解番号はAndroid版と共通です。正解選択肢はWeb版の画面表示時にだけシャッフルし、元JSONは変更しません。

現在のWeb版は99作品 / 608問を読み込みます。小1〜小3は各80問、小4は122問です。思考力強化サンプル教材は、最難関中学レベルを意識した完全オリジナル問題で、実在の過去問は使用していません。

## 読み上げ機能

Web版ではブラウザ標準の読み上げ機能を使います。本文、問題、選択肢、解説、通知表を読み上げできます。

読み上げに対応していない端末では、画面に「この端末では読み上げ機能を使えません。」と表示し、アプリはそのまま使えます。

## 学習履歴

解答履歴、テスト結果、学習通知表、読み上げ設定、先生コメント表示設定、最後に開いた学年はブラウザ内の保存領域に保存します。ブラウザのデータを削除すると履歴も消えます。

入力した名前・ニックネームもブラウザ内だけに保存します。外部送信は行わず、本名でなくても利用できます。設定画面からいつでも変更・削除できます。

## Android版との関係

このWeb版はAndroidネイティブアプリとは別の実装です。Android側のKotlinコード、AndroidManifest、教材JSONの元データは変更しません。

## 子供本位の学習設計

- PWA版にも基礎復習コースを追加しました。5問だけ、制限時間なしで、内容・順番・気持ち・根拠をゆっくり確認します。
- 結果画面に今日のがんばり、よく読めました、根拠みつけ名人などのバッジを表示します。
- 先生コメントは、点数だけでなく、本文に戻ること、もう一度考える姿勢、最後まで取り組むことを肯定します。
- 学習通知表では、今回よかったところと家庭での声かけも表示し、点数だけで判断せず次の学習につなげます。
- 初回オンボーディングは3画面だけ表示し、完了後はブラウザ内の保存領域に記録します。
- 「どれから始める？」から、やさしい読解、基礎復習、定期テスト、論理読解、思考力強化、保護者レポートへ直接移動できます。
- 保護者レポートは、今回よかったところ、がんばった形跡、次に伸ばしたい力、おすすめコース、家庭での声かけのカードに分けて表示します。
- 学びの木は、本文に戻る・考え直す・読み上げで確かめるなどの行動も成長として記録します。スキル別の進み具合と成長0〜10は、ブラウザ内の保存領域にだけ記録します。
- 小1〜小2では「本文をきく」などのやさしい読み上げボタンと、少しゆっくりした通常速度を使います。
- 小1〜小4の本文は、必要な語だけルビを付けて表示します。読み上げではルビのHTMLタグを読みません。
- 保護者レポートと学習通知表では、スキル別のようす、得意な力、次に練習したい力を外部ライブラリなしの横棒で確認できます。
